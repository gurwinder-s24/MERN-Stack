px means pixels
50% means 50% of the parent element
50vw means 50% of the viewport width
50vh means 50% of the viewport height
50vmin means 50% of the smaller of the viewport width or height
50vmax means 50% of the larger of the viewport width or height
50em means 50 times the current font size( means 50 into parent element font size)
50rem means 50 times the root element's font size
